# 🛡️ NewsGuard

**AI-Powered Fake News Detection & Credibility Analysis Platform**

NewsGuard helps users identify misinformation by analyzing news articles and generating credibility reports using Artificial Intelligence and Natural Language Processing (NLP). Users can submit either a news article URL or article text to receive detailed insights about its reliability.

---

## 🚀 Features

* 🔍 URL-based News Analysis
* 📝 Text-based News Analysis
* 🤖 AI-Powered Credibility Assessment
* 📊 Credibility Score Generation
* ⚠️ Fake News Probability Detection
* 🎯 Bias & Sentiment Analysis
* 🔐 Google Authentication
* 📈 Interactive Dashboard
* 🌙 Light & Dark Mode Support
* 📱 Fully Responsive Design

## ⚙️ How It Works

1. User enters a news article URL or article text.
2. NewsGuard extracts and processes the content.
3. AI analyzes the article for:

   * Credibility
   * Authenticity
   * Bias
   * Misinformation indicators
4. The platform generates:

   * Credibility Score
   * Fake News Probability
   * Risk Level
   * Detailed Explanation
5. Results are displayed through an intuitive dashboard.

---

## 📊 Key Outputs

* Credibility Score (0–100)
* Fake News Probability
* Risk Level (Low, Medium, High)
* Bias Detection
* AI-Generated Analysis Report

## 🔧 Installation

```bash
git clone https://github.com/your-username/newsguard.git
cd newsguard
npm install
npm run dev
```

### Environment Variables

Create a `.env` file and add:

```env
VITE_SUPABASE_URL=your_supabase_url
VITE_SUPABASE_PUBLISHABLE_KEY=your_supabase_key

OPENAI_API_KEY=your_openai_key
# or
GEMINI_API_KEY=your_gemini_key
```
